# Interchangeable Background Color

A tiny, focused web demo that lets you quickly swap the page background color using plain HTML, CSS and JavaScript.

A simple experiment — open the page, click a button, and the background color changes.

Highlights
- Minimal single-page demo (no build tools required).
- Clean, easy-to-follow HTML/CSS/JS for learning or reusing the idea.

Getting started
1. Clone or download the repository.
2. Open `index.html` in your browser (double-click or use a local server).

Usage
- Click the buttons on the page to change the background color.

Files
- `index.html` — page markup and controls.
- `style.css` — basic styling for the demo.
- `script.js` — JavaScript that handles color changes and any persistence.
- `Screenshot 2024-05-04 223652.png` — demo screenshot shown above.

How it works (quick overview)
- The page contains buttons or controls wired up in `script.js`.
- When a control is clicked, the script updates the document background color and may store the selection (e.g., localStorage) so the choice persists across reloads.

Customizing
- Edit `style.css` to change the layout and button styles.
- Edit `script.js` to add, remove, or change colors and persistence behavior.
